O presente projeto tem o objetivo de entretenimento educacional utilizando assuntos de banco de dados para a fixação do conteúdo.

Para o uso do código na sua máquina deve ser realizado o download gratuito da IDE oficial do Google, o Android Studio.

Através do link: https://developer.android.com/studio

Em seguida deve-se realizar o download do código no GitHub, descompactá-lo no seu computador, abrir o Android Studio e seguir os passos:
Clique em: File -> New -> Import Project

Localize a pasta onde descompactou o projeto e clique em OK

Com o projeto aberto no seu computador é possível compilar e rodar o programa através do menu: Run -> Run ‘Multiplayer’, ou clicando no atalho Shift+F10.

Para executar o aplicativo é necessário ter um emulador android instalado na sua máquina (o android studio auxilia neste processo) ou ter um aparelho Android e haver habilitado a função de Depuração USB no mesmo (como este passo varia de aparelho para aprelho, é aconselhável pesquisar a forma correta de realizar no seu).
